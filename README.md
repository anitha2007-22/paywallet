# 💳 PayWallet — FinTech Digital Wallet

A full-stack digital wallet application built with React, Node.js, PostgreSQL, and Socket.IO.

## 🌐 Live Demo
[https://paywallet-z99d.vercel.app](https://paywallet-z99d.vercel.app)

## ✨ Features
- 🔐 User Authentication (Register, Login, JWT)
- 💳 Digital Wallet (Add Money, Withdraw)
- 💸 Send & Receive Money
- 📋 Transaction History with Search & Filter
- 📊 Analytics Dashboard (Charts)
- 🔔 Real-time Notifications (Socket.IO)
- 🚨 Fraud Detection System
- 🛡️ Admin Dashboard
- 🌙 Dark / Light Mode
- 📱 Mobile Responsive

## 🛠️ Tech Stack
**Frontend:** React.js, Vite, Tailwind CSS, Chart.js, Socket.IO Client

**Backend:** Node.js, Express.js, PostgreSQL, Socket.IO, JWT

**Deployed on:** Vercel (Frontend) + Render (Backend + Database)

## 🚀 Local Setup

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/paywallet.git
cd paywallet
```

### 2. Setup Backend
```bash
cd server
cp .env.example .env
npm install
npm run dev
```

### 3. Setup Frontend
```bash
cd client
cp .env.example .env
npm install
npm run dev
```
